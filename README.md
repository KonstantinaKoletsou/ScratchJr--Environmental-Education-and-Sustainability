# ScratchJr--Environmental-Education-and-Sustainability
Ψηφιακό μαθησιακό περιεχόμενο το οποίο έχει αναπτυχθεί με την πλατφόρμα  οπτικού προγραμματισμού ScrtachJr για τη διδασκαλία περιβαλλοντικών θεμάτων στο νηπιαγωγείο. Οι θεματικές στις οποίες επικεντρώνεται είναι η "Ανακύκλωση" , "Η εξοικονόμηση νερού" και "Η προστασία των μελισσών".
